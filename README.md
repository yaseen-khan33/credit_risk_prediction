# 💳 Credit Risk Prediction (Loan Approval Prediction)

## 📌 Project Overview

This project is part of my Data Science & Analytics Internship. The objective is to build a machine learning model that predicts whether a loan application will be approved or rejected based on applicant details.

---

## 🎯 Objective

To predict loan approval status (**Loan_Status**) using applicant information such as income, education, employment status, and credit history.

---

## 📂 Dataset

* Source: Kaggle Loan Prediction Dataset
* File used: `train.csv`
* Target Variable: `Loan_Status`
